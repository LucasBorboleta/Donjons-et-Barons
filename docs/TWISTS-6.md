# TWISTS-6

## Introduction

Ce mémo recueille une nouvelle vague d'idées de "twists".

## Nouvelles mécaniques

### Evénements historiques

Sur dé favorable, et peut-être même sans ajouter de coût de défausse de cartes tactiques:

- **Peste** dit aussi "**Malédiction Noire**" : Piocher 3 cartes tactiques dont les positions indiquent les tuiles dont les **troupes** sont envoyées en **défausse**.
- **Révolte** dit aussi "**Vent de Sédition**" : Piocher 3 cartes tactiques dont les positions indiquent les tuiles dont les **troupes** sont envoyées en **réserve**.
- **Séisme** dit aussi "**Le Courroux de la Terre**" : Piocher 3 cartes tactiques dont les positions indiquent les tuiles dont les **donjons** sont envoyées **détruits**.
